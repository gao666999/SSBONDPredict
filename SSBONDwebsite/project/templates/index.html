This server predicts


disulfide bonds

candidates for given proteins.


  • This webserver takes a protein structure and predicts the potential mutation sites that may facilitate the formation of disulfide bonds.
  • The result is summarized in a table composed of amino acid pairs with the corresponding probability of forming a disulfide bonds, and the associated energy changes due to the mutation.
  • To use this predicition tool, you only need to upload a pdb file or input a PDB id. The results will be displayed in an interactive visual display powered by Jmol.
About Prediction Method

The algorithm is based on image classification methods in machine learning by utilizing the statistical information in the existing structures. The atomic coordinates of neighboring amino acids are used to compute the probability of forming di-sulfide bonds.
